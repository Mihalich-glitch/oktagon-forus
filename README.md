Учебный проект RESTful API для книжного магазина, построенный на **FastAPI**, **SQLAlchemy 2.0** и **PostgreSQL**. Проект включает в себя управление категориями и книгами с полной валидацией данных через **Pydantic**.

## Технологии
* **Python 3.10+**
* **FastAPI**: Современный и быстрый фреймворк для создания API.
* **PostgreSQL**: Реляционная база данных.
* **SQLAlchemy**: Мощная ORM для работы с БД.
* **Pydantic V2**: Валидация данных и схем.
* **Uvicorn**: ASGI-сервер для запуска приложения.

---

### 1. Подготовка базы данных
Убедитесь, что у вас установлен PostgreSQL. Создайте базу данных вручную:
```sql
CREATE DATABASE octagon_db;
```

### 2. Настройка окружения 
Склонируйте репозиторий и создайте файл .env в корневой папке проекта:
```t
DB_HOST=localhost
DB_PORT=5432
DB_NAME=octagon_db
DB_USER=ваш_логин
DB_PASSWORD=ваш_пароль
```

### 3. Установка зависимостей
Рекомендуется использовать виртуальное окружение:
```Bash
python -m venv venv
source venv/bin/activate  # Для Linux/Mac
# venv\Scripts\activate  # Для Windows

pip install -r requirements.txt
```

### 4. Инициализация базы данных
Запустите скрипт для автоматического создания таблиц и наполнения их тестовыми данными:
```Bash
python -m app.init_db
```

### 5. Запуск сервера
Запустите приложение с помощью Uvicorn:
```Bash
uvicorn app.main:app --reload
```
